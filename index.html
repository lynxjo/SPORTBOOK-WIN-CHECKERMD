Catatan:
Jika menang full, hitungan menjadi sama dengan odds.
Jika menang setengah, hitungan menjadi (Odds - 1) / 2 + 1.
Jika kalah setengah, hitungan menjadi 0.5.
Jika seri, hitungan menjadi 1 atau tidak mengubah perkalian.
Rumus Mix Parlay:
Hasil Odds = perkalian semua hasil odds partai.
Total Return = Hasil Odds × Nilai Bet.
Hasil Akhir = Total Return - Nilai Bet.
Jika hasil minus, berarti kalah sejumlah angka tersebut. Jika plus, berarti menang sejumlah angka tersebut.
